# SheetJS Kütüphanesi

Bu klasöre SheetJS (xlsx.full.min.js) kütüphanesini eklemeniz gerekiyor.

## İndirme

1. https://cdn.sheetjs.com/xlsx-0.18.5/package/dist/xlsx.full.min.js adresine gidin
2. Dosyayı indirin
3. Bu klasöre `xlsx.full.min.js` olarak kaydedin

## Alternatif

CDN'den otomatik indirilmesi için:

```bash
curl -o extension/libs/xlsx.full.min.js https://cdn.sheetjs.com/xlsx-0.18.5/package/dist/xlsx.full.min.js
```

veya

```bash
wget -O extension/libs/xlsx.full.min.js https://cdn.sheetjs.com/xlsx-0.18.5/package/dist/xlsx.full.min.js
```
